# Security Policy

## Supported Versions

`gantry` is pre-1.0. Only the latest `v0.x` release line receives security fixes.

| Version       | Supported |
| ------------- | --------- |
| latest `v0.x` | ✅        |
| older         | ❌        |

## Reporting a Vulnerability

Please report security issues privately via GitHub's
[private vulnerability reporting](https://github.com/RomanAgaltsev/gantry/security/advisories/new)
("Report a vulnerability" in the **Security** tab). Do not open public issues for security
reports.

We aim to acknowledge reports within 5 business days.
